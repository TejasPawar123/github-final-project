# This is the README.md file for the **github-final-project**

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

## Input:

* **p**: principal amount
* **t**: time period in years
* **r**: annual rate of interest

## Output:

* **simple interest = p * t * r / 100**

## Description

This project is a simple Bash-based calculator that computes simple interest using user inputs. It is designed as a beginner-friendly open-source project to demonstrate basic GitHub workflows, including repository creation, documentation, licensing, and contribution guidelines.

## Features

* Takes user input from terminal
* Calculates simple interest instantly
* Easy-to-understand Bash script
* Open-source under Apache License 2.0
